BBSBTM - Final Project CSCI E-15
Baseball Softball Team Manager is a web application designed for people who want to manage their baseball or softball team.

Version 1.0.0

- Demo Scorecard
- Log In - Log Out
- Create Account
- Edit Profile
- View Profile
- Create Team
- Create Players
- Create Score Card
- Save Score Card
- Close Score Card to come back to it later.
